# Physico-chemical process

Description: An ontology of physico-chemical processes, i.e. physico-chemical changes occurring in course of time.

More information can be found at http://obofoundry.org/ontology/rex

## Versions

### Stable release versions

The latest version of the ontology can always be found at:

http://purl.obolibrary.org/obo/rex.owl

(note this will not show up until the request has been approved by obofoundry.org)

### Editors' version

Editors of this ontology should use the edit version, [src/ontology/rex-edit.owl](src/ontology/rex-edit.obo)
